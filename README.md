This project features my naive solutions for the [Advent of Code 2016](https://adventofcode.com/2016) in Rust.
They are simple implementations designed to practice Rust programming.

## Project Structure

- `src/`: Source code.
- `inputs/`: Inputs for the problems.
Place your inputs in `inputs/n/input`, where `n` is the day number.

## Usage

Clone this repository and run:

```bash
cargo run --release
# or
cargo run
```

Or run the tests

```bash
cargo test
```
